Toggle Switch¶

  • $A \dashv B, B \dashv A$
  • We can take a boolean approximation where 0 is 'low', 1 is 'high':
A B stability
0 0 unstable
0 1 stable
1 0 stable
1 1 unstable

Two dimensions will never produce oscillatory behaviour - this is actually a theorem of Poincare. However, when we do the repressilator, this may come in useful.

N.B. this boolean approximation is not rigourous, it is not even often correct! But it can be useful.

Suppose we have the following situation:

  • $A \dashv B\text{-gfp} \dashv A$
  • $\text{heat} \dashv (A \dashv B)$, which is to say, heat represses the repression, and acts like a 'reset'
  • $\text{IPTG} \dashv (B \dashv A)$, IPTG is just some small molecule, used in the review

To switch from the high $B\text{-gfp}$ stable state to the $A$ state, we need to add IPTG.

This is behaving like a very simple memory module, the cells can 'remember' if it has seen IPTG, and that memory is contained within the $A$ state. Even if the IPTG is removed, the cells will remember that they have seen IPTG and remain in the $A$ state.

This toggle switch was constructed using components that had never seen each other before - and was one of the first papers in synthetic biology.

Dimensionless Equations¶

Dimensionless equations are good in that they focus your attention on the essential mathematical features. They are bad in that the connection to experiments can be obscured and the parameters might be harder to interpret.

  1. $$\dot{u} = \frac{\alpha_1}{1 + v^\beta} - u$$
  1. $$\dot{v} = \frac{\alpha_2}{1 + u^\gamma} - v$$

Questions¶

  1. How does this capture the behaviour of a toggle switch? When $u$ is high, $\dot{v}$ is small/ negative, and vice versa.

  2. What are the effective lifetimes (i.e. half-lives) of $u$ and $v$? They are actually the same - this is an assumption made by the paper! We can see this because if you were to ignore the first terms, both are just exponential decay with rate constant of 1, i.e. the same exponential decay for both. If you wanted different rates for both you would need to add another parameter in one of the equations.

  3. How have the quantities been normalised? Time has been normalised so that 1 unit of time corresponds to the effective lifetime (i.e. decay constant) of $u$ and $v$. Note that we were previously calling this decay constant $\alpha$, and was governed by $\alpha_{\text{deg}}$ and $\alpha_{\text{dil}}$. $u$ and $v$ have also been normalised so that $u=1$ corresponds to $K_d$ of $v$ (and vice versa). The $\alpha$'s have also been fudged by a similar factor. We can see this because what used to be called $K_d$ in Michaelis Mentens/ other contexts is now just 1.

  4. Which parameters might change if you increase the degredation rate (of both $u$ and $v$)? Both $\alpha_1$ and $\alpha_2$ will change. The exponents will not change, they govern the cooperativity. The $\alpha$'s will actually go down if you increase the degredation rate - you are dividing by a bigger number.

Stability Analysis¶

  1. Have $\dot{x} = ax$. When is $x=0$ stable? The answer is iff $a < 0$. Certainly $x=0$ is a fixed point in all these equations. A plot is helpful here! Recall 1a differential equations. Note that for discrete maps, the condition is for $|a| < 1$, but we are looking at a differential equation, not a difference equation.
  1. In general, we need all of the (real parts of the) eigenvalues of the Hessian matrix to be less than 0. This is described by a vector equation $\mathbf{\dot{x}} = \mathbf{Ax}$, and we need $Re(\lambda_i) < 0$ for all eigenvalues $\lambda_i$.
